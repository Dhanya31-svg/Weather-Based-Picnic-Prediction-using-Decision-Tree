# 🌳 Weather-Based Picnic Prediction using Decision Tree

## 📌 Overview

This project demonstrates an end-to-end Machine Learning workflow using the **Decision Tree Classification** algorithm to predict whether a picnic should be conducted based on weather conditions. The notebook covers the complete ML pipeline, from data generation and preprocessing to model training, evaluation, visualization, and prediction.

---

## 🎯 Problem Statement

Outdoor event organizers often face uncertainty due to changing weather conditions. Making the wrong decision can lead to financial losses and poor customer experience.

This project builds a Decision Tree model to predict whether a picnic should be **conducted** or **cancelled** based on weather-related features.

### Target Variable

- **1** → Picnic Can Be Conducted
- **0** → Picnic Should Be Cancelled

---

## 📊 Dataset

A realistic synthetic weather dataset was created for this project.

### Features

- Temperature (°C)
- Humidity (%)
- Windy
- Rainfall (mm)
- Cloud Cover
- Weather Score
- Pleasant Temperature
- Heavy Rain
- Good Weather
